# To Do List

Vue 3 • Composition API • Vite • JavaScript • LocalStorage

SPA-приложение для управления списком задач с использованием Vue 3 Composition API. Реализованы добавление, редактирование, удаление задач и сохранение данных в localStorage.

## О проекте

Это простое приложение для ведения списка задач.
Пользователь может добавлять новые задачи через форму, изменять их содержимое и отмечать задачи как выполненные.

Все задачи сохраняются в localStorage, благодаря чему данные не пропадают после перезагрузки страницы.

## Функциональность
+ добавление новой задачи
+ редактирование существующей задачи
+ удаление задачи
+ изменение статуса задачи (выполнена / не выполнена)
+ сохранение задач в localStorage
+ простая и понятная структура интерфейса

## Технологии

## Как запустить проект

```
git clone https://github.com/sofia-kildyaeva/ToDoList.git

cd ToDoList

npm install

npm run dev
```

После запуска приложение будет доступно по адресу: http://localhost:3000/
